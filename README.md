# Olá, eu sou o Thiago Somense! 👋

Mapeador de processos, gestor e desenvolvedor focado em criar soluções tecnológicas que unem o ambiente digital à eficiência operacional do mundo físico. Atuo fortemente no desenvolvimento de sistemas internos (ERP/WMS), aplicações web/mobile e na automação de processos industriais.

---

### 🛠️ O que eu faço

- **Full-Stack & Mobile Development:** Construção de aplicações web, desktop e aplicativos mobile robustos, modulares e escaláveis para gestão e experiência do usuário.
- **Automação Industrial:** Integração de software com maquinário, trabalhando com lógica de inversores, servo drives e geração/manipulação de G-code.
- **Arquitetura de Dados:** Modelagem de bancos de dados relacionais voltados para controle de estoque dinâmico (SKUs com variações), produção e logística.
